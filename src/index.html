Initiative Finder

The Initiative Finder app, a collaborative endeavor among a four-person team, stands as a robust Pathfinder initiative tracker. Engineered with features enabling seamless unit management, battle staging, and real-time tracking, it simplifies Pathfinder gameplay complexities. Leveraging Firebase for secure authentication and database management and styled with Material-UI, this app delivers a user-friendly interface. Its streamlined functionalities empower Game Masters, allowing for focused storytelling. This project showcases our ability to merge technical expertise with intuitive design, delivering an indispensable tool tailored for Pathfinder enthusiasts.

Technologies Used: React, MaterialUI, Firebase, JavaScript, HTML, CSS

MaraudersMemoir

MaraudersMemoir is a Harry Potter search engine that uses external data sources to provide information on characters and spells. It incorporates React Query to offer infinite scrolling for an efficient user experience. Moreover, the app includes a Quote of the Day feature that uses local storage to provide daily inspiration from the Harry Potter series. This project was created to showcase combining data with modern web technologies to create a user-friendly and engaging experience for Harry Potter enthusiasts.

Technologies Used: React, MaterialUI, ReactQuery, JavaScript, HTML, CSS

ShelfMate

ShelfMate is a pantry inventory management system designed to streamline item tracking and management. Users can easily add, edit, and delete items in their inventory. It includes secure user authentication with bcrypt-encrypted passwords, utilizes PostgreSQL and Sequelize for database management. To enhance user convenience, the system employs session stores and cookies, ensuring data persistence throughout user sessions.

Technologies Used: PostgreSQL, Sequelize, Express.js, Node.js, Tailwind CSS, Bcrypt, Session Stores, Cookies
